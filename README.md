# Lina

Assistente para pesquisa semântica, organização e enriquecimento de notas Markdown.

**Versão:** 0.1.0  
**Autor:** Anselmo Pinheiro  
**Licença:** MIT

## Funcionalidades (futuras)

- Pesquisa semântica em notas Markdown
- Sugestões de YAML, tags e links internos
- Sugestão de pasta de destino
- Integração com Ollama
- Integração futura com OpenRouter

## Instalação

1. Copiar a pasta `lina` para `.obsidian/plugins/` do vault.
2. Ativar o plugin nas definições do Obsidian (Plugins da Comunidade).

## Desenvolvimento

```bash
npm install
npm run build
```

O ficheiro `main.js` será gerado na raiz do projeto.
